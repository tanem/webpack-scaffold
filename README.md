## run

Install dependencies:

```sh
$ npm install
```

Build for dev:

```sh
$ npm run build-dev
```

Or prod:

```sh
$ npm run build-prod
```

Then open in a browser:

```sh
$ open index.html
```

Try resizing the window to see various components in action.
